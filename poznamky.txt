1. Instalacia Django do python:
pip install Django
pip list

2. yvtvorenie django projektu:
django-admin startproject nazov_projektu .

3.nastavenie slovenciny pre projekt (mysite/settings.py):
LANGUAGE_CODE = 'sk'
TIME_ZONE = 'Europe/Bratislava'

4.spustenie vyvvojoveho serveru
python manage.py runserver
python.exe .\manage.py runserver

5. vytvorenie aplikacie
python manage.py startapp nazov_app

6. zaregristrovanie aplikacie do projektu (mysite/settings.py):
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'nazov_app',
]

7. vytvorenie jednoducheho views (skuska/views.py)
from django.shortcuts import render, HttpResponse

def index(request):
    return HttpResponse('Hello World!')

8. vytvorenie urlpattern:
8.1 mysite/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('skuska/', include('skuska.urls')),
]
8.2
- vytvorit subor urls.py v priecinku aplikacie (skuska)
- upravit urls.py:
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name = 'index'),
]

9.Template
9.1
- vytvorit v priecinku apliklacie priecinok templates/nazov_aplikacie
- v tomto priecinku vytvorit html subor (napr. index.html)
9.2 upravit views tak aby pouzival template
from django.shortcuts import render

def index(request):
sucet = 10+5
apar="cislo a je parne"
return render(request, 'kalkulacka/index.html',{"sucet":sucet,"apar":apar})



10.css, obrázky
-Vytvorit v priecinku aplikacie preičinok static/názov_aplikácie
-v tomto priecinku vytvorit css(naprikal kalkulacka.css)
-do tohto pirecinku nakopirovat obrazky
-do template(index.html) doplnit:
-uplne na zaciatok doplnit{% load static %}
-do hlavicky odkaz na style
< head >
< link rel= "stylesheet" href="{% static 'kalkulacka/kalkulacka.css' %}" >
